Base

Used to style the html tags

Example of Base File


html, body {...}
h1, h2, h3, h4, h5, h6 {...}
table {...}

Layout

Contains style for structure

Layout can include the following

  1. Grid System
  2. Style of constrain box that include modules. (ex: header, footer)

Naming convention for Layout classes


.l-section {...}
.layout-section {...}

Module

A component that can be reused on other page

Example of Modules:

  • Buttons
  • Tabs
  • Accordion
  • Dropdown
  • Slider
  • Testimonials
  • Modal
  • Others

State

Represent the current state of module

Naming convention for State


/* Global State */
.is-active {}
.is-disabled {}
.is-hidden {}

/* Module State */
.btn-is-disabled {}
.btn-is-enabled {}

/* Module State via data attribute */
.btn[data-state="disabled"] {}

Manipulate States


// For class state
$('.item').removeClass('is-hidden').addClass('is-active');

// For data attribute state
$('.item').data('state', 'active');
