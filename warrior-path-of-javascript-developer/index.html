Practice

Conway's Game of Life

About Game

Rules

Any live cell with fewer than 2 neighbours dies

Any live cell with 2 or 3 neighbours lives on

Any live cell with more than 3 neighbours dies

Any dead cell with exactly 3 neighbours becomes a live cell

Declare Variables

var rows = parseInt(sceneWidth / scaleCell);

Generate Grid


for (var x = 0; x < rows; x++) {
  grid[x] = [];
  for (var y = 0; y < cols; y++) {
    grid[x][y] = 0;
  }
}

Add cells to the grid


function seedGrid(cellsCount) {
  for (var i = 0; i < cellsCount; i++) {
    grid[getRandomNumber(0, rows)][getRandomNumber(0, cols)] = 1;
  }
}

Draw a cell


ctx.fillStyle = color || 'black';
